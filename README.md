# 📈 Food Delivery Time Prediction using Linear Regression

This is my first machine learning project where I built a **Linear Regression model** to predict food delivery time based on various factors like weather, traffic, and courier experience.

## 📖 Project Overview

The goal of this project is to predict **Delivery Time (in minutes)** using real-world features such as:

* Weather conditions
* Traffic levels
* Time of day
* Vehicle type
* Courier experience

This project also compares different regression models:

* Linear Regression
* Ridge Regression
* Lasso Regression

---

## 🧹 Data Preprocessing

Before training the model, the dataset was cleaned and prepared:

* Filled missing categorical values using **mode**
* Filled missing numerical values using **median**
* Converted categorical variables into numeric using **One-Hot Encoding**
* Dropped the target column (`Delivery_Time_min`) to create feature set

---

## ⚙️ Models Used

* **Linear Regression** (main model)
* **Ridge Regression** (regularization)
* **Lasso Regression** (feature selection)
